## 项目概述

本仓库包含了多种机器学习任务的实现，如分类、回归、聚类和神经网络模型。项目使用了流行的机器学习库，如 **Pytorch**来实现这些模型。

主要特性包括：
- 图像分类（例如，Dogs vs Cats 数据集）
- K-means 聚类
- 神经网络模型（例如，简单的神经网络分类任务）
- MNIST 数据集上的深度学习模型
